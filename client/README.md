# BunnyMarket 클라이언트

## 소개

BunnyMarket 클라이언트는 토끼 상품을 구매하고 판매할 수 있는 온라인 마켓플레이스입니다.

## 설치

1. 이 저장소를 클론합니다.
2. 터미널에서 `npm install` 명령어를 실행합니다.

## 사용법

1. 터미널에서 `npm start` 명령어를 실행합니다.
2. 웹 브라우저에서 `http://localhost:3000`으로 접속합니다.

## 기능

- 상품 목록 조회
- 상품 검색
- 상품 구매
- 상품 등록

## 기여

1. 이 저장소를 포크합니다.
2. 새로운 브랜치를 생성합니다.
3. 변경 사항을 커밋하고 푸시합니다.
4. 풀 리퀘스트를 생성합니다.
